\hypertarget{1}{%
\section{}\label{1}}
Prove that $e^{2πiF(x)} = f(e^{2πix})$, for all $x ∈ R$.

$F$ is a lift function of $f$ if
\begin{equation}
    \pi \circ F = f \circ \pi
\end{equation}
and
\begin{equation}
    \pi(x) = exp(2\pi i x)
\end{equation}
In our case:
\begin{equation}
F(x) = x + ω + \frac{\epsilon}{2 \pi} sin(2 \pi x)
\end{equation}

\begin{equation}
f_{ω, \epsilon}(\theta) = \theta + 2 \pi ω + \epsilon sin(\theta)
\end{equation}

\begin{equation}
e^{2πiF(x)} = e^{2\pi i (x+ \omega + \frac{\epsilon}{2 \pi} sin(2 \pi x))}
\end{equation}
\begin{equation}
f(e^{2πi x}) = e^{2\pi i x } + 2 \pi ω + \epsilon sin(e^{2\pi i x })
\end{equation}

to prove the lift we could solve analytically:

\begin{equation}
\begin{split}
    e^{2\pi i (x+ \omega + \frac{\epsilon}{2 \pi} sin(2 \pi x))} = e^{2\pi i x } + 2 \pi ω + \epsilon sin(e^{2\pi i x }) \\
    e^{2\pi i x} e^{2\pi i \omega} e^{i \epsilon sin(2 \pi x)} = e^{2\pi i x } + 2 \pi ω + \epsilon sin(e^{2\pi i x }) \\
    \dots
\end{split}
\end{equation}

\hypertarget{2}{%
\section{}\label{2}}Prove that $F(x + 1) = F(x) + 1$, for all $x ∈ R$.

\begin{equation} \label{eq:8}
\begin{split}
    F(x+1) = (x + 1) + \omega + \frac{\epsilon}{2 \pi} sin(2 \pi (x + 1))\\
    x + 1 + \omega + \frac{\epsilon}{2 \pi} sin(2 \pi x + 2 \pi)* \\
    x + 1 + \omega + \frac{\epsilon}{2 \pi} sin(2 \pi x)\\
\end{split}
\end{equation}
\begin{equation}
\begin{split}
    *sin(2 \pi x + 2 \pi) = sin(2 \pi x) cos(2 \pi) + sin(2 \pi) cos(2 \pi x)\\
    sin(2 \pi x + 2 \pi) = sin(2 \pi x) * 1 + 0 * cos(2 \pi x)\\
    sin(2 \pi x + 2 \pi) = sin(2 \pi x)\\
\end{split}
\end{equation}

\begin{equation} \label{eq:10}
\begin{split}
    F(x) + 1 = x + ω + \frac{\epsilon}{2 \pi} sin(2 \pi x) + 1\\
    x + 1 + ω + \frac{\epsilon}{2 \pi} sin(2 \pi x)
\end{split}
\end{equation}

As we can see, Equation \ref{eq:8} is equal to Equation \ref{eq:10}, hence proving that $F(x + 1) = F(x) + 1$.

\hypertarget{3}{%
\section{}\label{3}}Choose (your favourite) number $0 < \epsilon < 1$, we denote this number by $\epsilon_{0}$, and compute numerically an approximation of the rotation number $\rho(f_{ω, \epsilon_{0}})$

\begin{equation*}
\rho(f_{ω, \epsilon_{0}}) = \lim_{n\to\infty} \frac{F^{n}_{\omega, \epsilon_{0}}(x)}{n}
\end{equation*}

To do this numerical computation, you can follow the following steps
\begin{itemize}
    \item Fix N > 1 and make a discretization of the parameter ω, so consider the discrete values
    \begin{equation}
        \omega_{k} = \frac{k}{N} \quad k = 0, \dots, N
    \end{equation}
    \item For each $\omega_{k}$ and your favorite $\epsilon_{0}$, compute
    \begin{equation*}
        \rho(f_{ω_{k}, \epsilon_{0}}) = \lim_{n\to\infty} \frac{F^{n}_{\omega_{k}, \epsilon_{0}}(x)}{n}
    \end{equation*}
\end{itemize}

The way to represent this function is in the x-axis you put the $\omega_{k}$ values and in the y-axis you plot the corresponding
rotation number $\rho(f_{\omega_{k}}, \epsilon_{0})$.
